
# 🎵 Music Recommendation System
A Machine Learning-based Music Recommendation System built using Python and Streamlit.
This project suggests songs similar to a given input using cosine similarity.

🚀 Features
* 🎧 Recommend songs based on similarity
* 📊 Uses real Spotify dataset
* ⚡ Fast and interactive web interface
* 🌐 Deployed using Streamlit

🛠️ Tech Stack
* Python
* Pandas
* Scikit-learn
* Streamlit

📂 Dataset
Spotify Tracks Dataset (CSV file)

⚙️ How It Works
1. Data preprocessing using MinMaxScaler
2. Feature selection from dataset
3. Cosine similarity calculation
4. Recommend top similar songs

💻 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
